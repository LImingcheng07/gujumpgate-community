## 使用建议

当前仍然更建议：

- 优先使用 `导出至 - SESSION JSON 导入`

原因很直接：OAuth 风控仍然重，很多时候会跳手机号绑定，SESSION 链路反而更现实。

## 安装方式

1. 下载本页附件里的压缩包
2. 解压到本地目录
3. 打开 `chrome://extensions/`
4. 开启开发者模式
5. 选择“加载已解压的扩展程序”
6. 选中解压后的目录
7. 按 README 教程填写你自己的配置
